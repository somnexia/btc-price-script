📊 Bitcoin Price Tracker (Python)

Simple Python script that fetches real-time Bitcoin price data from a public API and processes it for further use or analysis.

The project is designed for learning purposes and demonstrates how to work with external APIs, handle JSON responses, and manage dependencies using a requirements.txt file.

⚙️ Features
Fetches current Bitcoin price from a public API
Parses and processes JSON responses
Lightweight and easy to extend
Dependency management via requirements.txt
🧰 Tech Stack
Python 3.x
Requests (for API calls)
📦 Installation
pip install -r requirements.txt
🚀 Usage
python main.py
🧠 Notes

This project is intentionally simple and serves as a base for:

data collection scripts
trading bots (if you enjoy pain and volatility)
analytics pipelines
